
Matplotlib is easy to use and an amazing visualizing library in Python. 
It is built on NumPy arrays and designed
to work with the broader SciPy stack and 
consists of several plots like line, bar, scatter, histogram, etc. 



Seaborn is a library mostly used for statistical plotting in Python. 
It is built on top of Matplotlib and provides beautiful default styles 
and color palettes to make statistical plots more attractive
